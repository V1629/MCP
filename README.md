# Zangetsu: Web Automation Agent

> ⚠️ **Under Active Development**

An advanced automation platform for browser automation, web scraping, and task automation.

## Features
- **Multi-Browser Support**: Chrome, Firefox, Brave (in progress)
- **Web Navigation & Form Automation**: Basic implementation complete
- **Social Media Integration**: LinkedIn (in development)
- **File Management & Search**: In progress

## Technologies
Python, langchain_groq, mcp, Playwright, Selenium

## Quick Start
```bash
git clone https://github.com/V1629/mcp.git
cd mcp
pip install -r requirements.txt
```

## Project Structure
```
mcp/
├── browser/          # Browser automation
├── file_operations/  # File handling
├── search/          # Search features
├── utils/           # Utilities
└── main.py         # Entry point
```
